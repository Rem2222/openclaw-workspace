## ⚙️ BEADS ИНТЕГРАЦИЯ

**Важно:** Используй Beads для управления задачами.

### Команды:
```bash
bd ready                    # показать доступные задачи
bd create "описание | File: путь.md"  # создать задачу
bd show <id>               # посмотреть детали
bd update <id> --claim      # взять в работу
bd close <id>              # закрыть задачу
```

### Для тестировщика:
- После получения задачи: `bd update <id> --claim` — взять задачу
- После каждого этапа тестирования: `bd create "[Тест] Backend API | File: Отчёт_тестирования.md"` — создать подзадачу
- После завершения: `bd close <id>` — закрыть задачу
- Если найден баг: `bd create "[Bug] Описание | File: Отчёт_тестирования.md#bug-1"`
