## OpenViking Search — поиск по контекстной базе знаний

Использует OpenViking API для семантического поиска по файловой системе знаний с уровнями контекста L0/L1/L2.

### Когда использовать
- Пользователь спрашивает о прошлых решениях, контексте, уроках
- Нужен семантический поиск с re-ranking
- Требуется визуализация пути поиска

### API OpenViking (localhost:1933)

**POST /api/v1/search** — поиск с контекстом
```json
{
  "query": "вопрос пользователя",
  "level": 1,  // 0=абстракция, 1=обзор, 2=полное
  "top_k": 10,
  "filters": {}
}
```

**GET /api/v1/files/{path}** — чтение файла

**GET /api/v1/browse/{path}** — просмотр директории

### Уровень контекста
- **L0** — максимальная абстракция, минимум токенов
- **L1** — обзорный уровень (по умолчанию)
- **L2** — полное содержание

### Пример использования
```bash
# Поиск через curl
curl -X POST http://localhost:1933/api/v1/search \
  -H "Content-Type: application/json" \
  -d '{"query": "как настроить ollama", "level": 1, "top_k": 5}'
```

### Интеграция с OpenClaw
- Хранит знания в ~/.openviking/workspace
- Поддерживает гибридный поиск (семантический + ключевые слова)
- Возвращает cited results с источниками
