# SKILL.md

## OpenViking Info — просмотр структуры и статистики

Показывает информацию о структуре базы знаний, статистике файлов и содержимом.

### Когда использовать
- Пользователь спрашивает "что у меня есть"
- Нужна статистика по файлам
- Требуется обзор структуры

### Функции

1. **Структура workspace** — дерево файлов и папок
2. **Статистика** — количество файлов, размер, даты
3. **Содержимое** — краткий обзор файлов
4. **Последние изменения** — свежие записи

### Примеры использования

```bash
# Структура workspace
node skills/openviking-info/info.js structure

# Статистика файлов
node skills/openviking-info/info.js stats

# Последние изменения
node skills/openviking-info/info.js recent

# Обзор MEMORY.md
node skills/openviking-info/info.js memory
```

### Вывод

- Текстовый формат для консоли
- JSON для программной обработки
