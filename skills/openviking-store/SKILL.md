# SKILL.md

## OpenViking Store — хранение информации в контекстной базе

Сохраняет знания, заметки и контекст в файловую систему OpenViking для последующего поиска.

### Когда использовать
- Пользователь просит запомнить что-то
- Нужно сохранить результат разговора
- Требуется создать новую заметку или документ

### Структура хранения

```
~/.openclaw/workspace/
├── MEMORY.md              # Долгосрочная память (кураторная)
└── memory/
    ├── YYYY-MM-DD.md      # Дневники по дням
    ├── YYYY-MM-DD-*.md    # Сессии и события
    └── topics/            # Тематические папки
        ├── projects.md
        ├── learnings.md
        └── ...
```

### Типы хранения

1. **Дневник дня** — `memory/YYYY-MM-DD.md`
   - События текущего дня
   - Авто-создаётся при первом записи

2. **Долгосрочная память** — `MEMORY.md`
   - Значимые решения
   - Уроки и паттерны
   - Контекст пользователя

3. **Тематические файлы** — `memory/topics/*.md`
   - Проекты
   - Изучаемые темы
   - Справочная информация

### API

```javascript
// Сохранение в дневник дня
await storeInDailyLog(title, content);

// Сохранение в долгосрочную память  
await storeInMemory(section, content);

// Создание тематического файла
await storeInTopic(topic, title, content);
```

### Пример использования

```bash
# Сохранение заметки
echo "# Заметка\n\nТекст заметки" >> memory/2026-03-16.md

# Обновление MEMORY.md
node skills/openviking-store/store.js memory "Уроки" "Новый урок"
```
