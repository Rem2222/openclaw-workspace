### Главная проблема (НЕ решена)
**MiniMax-M2.7 не выводит `tool_calls`** — выводит текст.

JAWL использует OpenAI function calling API (`tools=[]`, `tool_choice`), но MiniMax-M2.7 через OpenAI-compatible endpoint игнорирует это и пишет обычный текст в content.

Симптомы:
- "Агент не вызвал инструменты" в каждом цикле
- Модель выдаёт 700-1500 токенов текста, но 0 tool_calls
- На чистом контексте первые 1-2 шага работают, потом перестают

Варианты решения:
1. GLM-4.7 — но MiniMax.io не поддерживает (нужен Z.AI API)
2. MiniMax-Text-01 — может иметь лучший function calling
3. Переключить ReAct-loop на text-based JSON parsing вместо tool_calls
