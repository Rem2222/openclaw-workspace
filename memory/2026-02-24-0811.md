## 🆓 Бесплатные модели OpenRouter

### Как использовать бесплатные модели

OpenRouter предоставляет доступ к бесплатным моделям через специальный суффикс `:free`:

```
meta-llama/llama-3.2-3b-instruct:free
```

### Бесплатный роутер

OpenRouter также предоставляет **Free Models Router** — автоматический роутер, который выбирает бесплатную модель случайно из доступных:

```
openrouter/free
```

### Ограничения бесплатных моделей

1. **Rate limits** — ограничения на количество запросов в минуту/час
2. **Availability** — доступность может отличаться от платных версий
3. **Performance** — может быть медленнее платных моделей
4. **Context window** — может быть меньше, чем у платных версий

### Популярные бесплатные модели

Из документации OpenRouter можно выделить:

- **Meta Llama 3.2 3B Instruct** (`meta-llama/llama-3.2-3b-instruct:free`) — небольшая, но быстрая модель
- **Mistral Nemo** — также доступна в бесплатной версии
- **Разные версии Llama** — часто доступны бесплатно через OpenRouter

### Важные моменты

1. **Бесплатные модели не требуют API ключа** — можно использовать без добавления кредитов
2. **Ограничения по токенам** — есть лимиты на длину контекста
