- API требует **два** параметра:

2. Проверить что `api.minimax.io` резолвится с его Windows: `ping api.minimax.io`
3. Скинуть HMACCCS value

### Архитектура CodexBar

- `_CookieDecryptor` — читает Chromium cookies через DPAPI + AES-GCM (Chrome/Edge/Brave v10, НЕ v127+)
- Browser должен быть закрыт при чтении cookies — иначе "DB locked"
- MiniMaxDataFetcher: `_load_cookies_from_browser()` → проверяет Bearer токен, потом HMACCCS
- OpenCodeDataFetcher: парсит workspace HTML для usage data
