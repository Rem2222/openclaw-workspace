## Разговор: Модель для LCM суммаризации

**Вопрос Романа:** "кто команду дал" — почему LCM использует gpt-oss-20b для суммаризации

**Ответ:**
- LCM использует `openrouter/openai/gpt-oss-20b` как **дефолтную модель для суммаризации** из OpenClaw Core
- Это fallback когда не задан LCM_SUMMARY_MODEL
- Характеристики: contextWindow 131K, reasoning: true, стоимость $0.03 in / $0.14 out

**Можно изменить:**
- `export LCM_SUMMARY_MODEL=zai/glm-5` (или GLM-4.7)
- Или в конфиге plugins.entries.lossless-claw.config

**Источник информации:** ~/.openclaw/extensions/lossless-claw/README.md
