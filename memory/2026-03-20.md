### Критические баги исправлены:

#### 🔴 `.then()` handler с lambda не работает с async
```python
# ❌ Ошибка: lambda возвращает корутину, Gradio ждет 6 значений
).then(
    fn=lambda uri: open_resource_viewer(uri),  # возвращает корутину!
    ...
)

# ✅ Решение: async wrapper
async def open_resource_viewer_wrapper(uri: str) -> Tuple[...]:
    return await open_resource_viewer(uri)

).then(
    fn=open_resource_viewer_wrapper,  # теперь асинхронно
    ...
)
```
