### Heartbeat
- Запускается каждые **30 минут** по умолчанию
- Настраивается через `agents.defaults.heartbeat.every` в конфиге
- Можно отключить (`0m`) или изменить интервал

### memU — Фреймворк памяти

**Что изучали:** https://github.com/NevaMind-AI/memU

**Суть:**
- Фреймворк памяти для 24/7 проактивных агентов (OpenClaw, MoltBot, ClawDBot)
- Память как файловая система — структурированная, иерархическая
- Прогнозирует намерения пользователя до того, как он спросит
- Экономит токены через кэширование и embedding-only поиск

**Архитектура:**
```
memory/
├── preferences/
├── relationships/
├── knowledge/
└── context/
```

**Требования:**
- LLM (для извлечения памяти, понимания контекста)
- Embedding модель (для семантического поиска)
- Python 3.13+ (или Docker)

## Где остановились

**Следующие шаги для запуска memU локально:**

1. **Установить embedding модель:**
   ```bash
   ollama pull nomic-embed-text
   ```

2. **Проверить работу:**
   ```bash
   curl http://10.110.0.203:11434/api/embeddings \
     -H "Content-Type: application/json" \
     -d '{"model": "nomic-embed-text", "prompt": "test"}'
   ```

3. **Решить вопрос с Python 3.13:**
   - Вариант А: Обновить Python
   - Вариант Б: Использовать Docker
   - Вариант В: Использовать cloud API memU (`https://api.memu.so`)
