### 1c-batch — Простая автоматизация через Batch

**Автор:** Vladimir Kharin (Владимир Харин)
**GitHub:** https://github.com/vladimir-kharin/1c-batch
**Stars:** 59 ⭐ | **Forks:** 5
**Лицензия:** MIT
**Дата создания:** 7 февраля 2026

**Функционал (8 операций):**
1. Сборка обработки (XML → EPF/ERF)
2. Разборка обработки (EPF/ERF → XML)
3. Загрузка конфигурации (XML → база)
4. Выгрузка конфигурации (база → XML)
5. Загрузка расширения (XML → база)
6. Выгрузка расширения (база → XML)
7. Запуск предприятия
8. Запуск конфигуратора

**Требования:**
- Windows (Batch-скрипты)
- 1С:Предприятие 8.x

**Настройка:**
Создать файл `.1c-devbase.bat` с путём к 1С и параметрами базы
