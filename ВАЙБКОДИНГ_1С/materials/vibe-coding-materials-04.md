### ⚡️ MCP СЕРВЕРЫ — 7 000 ₽

**Формат:** 8 Docker контейнеров, подключаемых одной строкой

**Цитата:** "При правильной настройке заставляют ИИ 'творить чудеса' при кодинге в 1С"

**Список серверов:**
1. **Поиск (RAG+Fulltext) по коду** — поиск по коду конфигурации
2. **Поиск (RAG+Fulltext) по справке** — справка и запросы
3. **Поиск (RAG) по шаблонам кода** — шаблоны для 1С + добавление своих
4. **Проверка синтаксиса BSL LS** — BSL Language Server
5. **Проверка кода** — 1С:Напарник (нужен ключ)
6. **Поиск по метаданным (Граф + Субагент)** — описание метаданных + поиск по бизнес-сущности
7. **Поиск (RAG+Fulltext) по функциям БСП** — любые версии
8. **Генерация форм (RAG + Субагент)** — для 1С

**Рекомендация:** Перед использованием рекомендуется пройти курс
