### 🆕 Курс по RAG (ИИ агенты в Enterprise) — ГОТОВ (28.02.2026)

**Ссылка:** https://vibecoding1c.ru/rag

**Статус:** Полностью доступен

**Цель:** Для тех, кто всерьёз собирается внедрить ИИ в компании

**Внутри:**
- Векторные базы: Milvus, Qdrant
- Не векторные: Elastic, Neo4j
- Работа с неструктурированными данными
- Локальные модели, vLLM, кластеры, оборудование
- Embedding-и, Chunk-и
- **Различные типы RAG:** GraphRAG, CRAG, LightRAG

**Результат:** Разработка RAG для ИИ агента с индексацией любых объёмов данных

**Доп. мотивация:**
- Понять, как разработать MCP "лучше чем у Олега"
- Разобраться с Neo4j, графами, почему Qdrant локально плохо, Chroma на сервере тоже плохо
- Понять векторы, как поиск возвращает статьи без похожих слов
- Чем embedding-модели отличаются от LLM

**Уникальность:** "Чуть ли не единственный источник, где системно всю эту инфу удалось собрать"

**Контекст:** RAG давно существует, но информация разрозненная (похоже с историей DWH)

**Цитата:** "Кажется в 2026-м не знать как работает RAG и не применять эту историю для ИИ это уже 'странно'"
