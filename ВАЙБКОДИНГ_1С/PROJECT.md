## 🎯 Цели обучения

1. **Понять** принципы вайбкодинга и agentic engineering
2. **Настроить** среду разработки (Cursor, MCP, модели)
3. **Освоить** инструменты для 1С (MCP-серверы, субагенты)
4. **Применить** в реальной работе (задачи из 1С)
5. **Создать** собственных агентов для автоматизации

## 📊 Уровни обучения (5 уровней)

### Уровень 0: Подготовка (1-2 дня)
**Цель:** Понять основы, мотивация, теория

### Уровень 1: Разминка (1-2 недели)
**Цель:** Настройка среды, первые шаги

### Уровень 2: Инструментарий (2-3 недели)
**Цель:** MCP-серверы, субагенты, фреймворки

### Уровень 3: Интеграция с 1С (1-2 месяца)
**Цель:** Практика в 1С, реальные задачи

### Уровень 4: Продвинутый (2-3 месяца)
**Цель:** SDD, оркестрация моделей, оптимизация

### Уровень 5: Enterprise (3-6 месяцев)
**Цель:** RAG, агенты для компании, OneAPA

## 🔄 Процесс работы

1. **Каждое сообщение** по проекту сохраняется в `docs/LESSONS.md`
2. **Новые материалы** добавляются в `materials/` или `sources/`
3. **Транскрипции** видео → `transcripts/`
4. **ROADMAP.md** обновляется по мере прогресса
5. **Вопросы Rem** → сохраняются, отвечаются, добавляются в контекст
