# OpenRouter Бесплатные Модели

## 🆓 Бесплатные модели OpenRouter

OpenRouter предоставляет доступ к бесплатным моделям, которые можно использовать без добавления кредитов.

## Как использовать

### 1. Бесплатный роутер (рекомендуется)

```
openrouter/free
```

**Алиас:** `free-router`

Это автоматический роутер, который выбирает бесплатную модель случайно из доступных. Он фильтрует модели по поддерживаемым функциям (изображения, инструменты, структурированные выводы).

### 2. Конкретные бесплатные модели

| Модель | ID | Алиас | Описание |
|--------|-----|-------|----------|
| Llama 3.2 3B Instruct (Free) | `meta-llama/llama-3.2-3b-instruct:free` | `llama-3b-free` | Небольшая, быстрая модель для базовых задач |
| Llama 3.2 1B Instruct (Free) | `meta-llama/llama-3.2-1b-instruct:free` | `llama-1b-free` | Самая маленькая модель, для простых задач |

## Ограничения бесплатных моделей

### ⚠️ Важные ограничения:

1. **Rate limits** — ограничения на количество запросов в минуту/час
   - Бесплатные модели имеют более строгие лимиты
   - Могут быть временные блокировки при высокой нагрузке

2. **Context window** — ограниченный контекст
   - Обычно 8K токенов (vs 128K+ у платных)
   - Подходит для коротких диалогов

3. **Performance** — производительность
   - Меньшие модели (1B-3B) слабее больших (70B+)
   - Могут быть медленнее в ответе

4. **Availability** — доступность
   - Зависит от нагрузки на провайдеров
   - Могут быть временно недоступны

5. **Функции** — ограничения по возможностям
   - Многие бесплатные модели не поддерживают:
     - Изображения (vision)
     - Инструменты (tool calling)
     - Структурированный вывод (JSON)
     - Длинный контекст

## Использование в OpenClaw

### Через команду сессии:
```
/session model openrouter/openrouter/free
/session model free-router
```

### Через алиасы:
```
/session model llama-3b-free
/session model llama-1b-free
```

### Для тестирования:
```bash
# Проверить доступные модели
openclaw status

# Переключиться на бесплатную модель
/session model free-router

# Сделать запрос
Привет! Как дела?
```

## Когда использовать бесплатные модели

### ✅ Подходит для:
- Тестирования и экспериментов
- Простых текстовых задач
- Коротких диалогов
- Обучения и разработки
- Низкобюджетных проектов

### ❌ Не подходит для:
- Сложных задач с изображениями
- Длинных контекстов (>8K токенов)
- Продакшен-приложений
- Задач, требующих высокой точности
- Интеграций с инструментами

## Стоимость

**Бесплатно!** Все модели с суффиксом `:free` или `openrouter/free` не требуют оплаты.

## Альтернативы

Если бесплатные модели недостаточно мощные, рассмотрите:

1. **MiMo V2 Flash** — бесплатная модель от Xiaomi (уже настроена в OpenClaw)
2. **Gemini 2.0 Flash-Lite** — бесплатная модель от Google
3. **DeepSeek Chat** — недорогая модель с хорошим качеством

## Проверка работы

После настройки проверьте, что модели доступны:

```bash
# Проверить статус OpenClaw
openclaw status

# Проверить доступные модели
openclaw models list
```
