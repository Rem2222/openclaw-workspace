# CodexBar v2.2.0 — Инструкция

## Что нового

**v2.2.0** перешёл на browser cookie authentication. OpenCode и MiniMax теперь автоматически читают cookies из твоего браузера (Chrome / Edge / Brave).

---

## Установка

1. Скачай `codexbar.py` с GitHub:
   https://github.com/Rem2222/CodexBar-Win-Enhance/blob/master/codexbar.py

2. Положи файл в папку и запусти:
   ```
   pythonw codexbar.py
   ```

   (или `start-codexbar.bat` если делал раньше)

---

## Настройка провайдеров

### OpenCode

1. **Залогинься в браузере** на https://opencode.ai
   - Подойдёт Chrome, Edge или Brave
   - Cookies читаются автоматически

2. Если нужна manual авторизация (если auto не работает):
   - Settings → OpenCode → вставь `auth` cookie вручную
   - Как получить cookie:
     1. Открой https://opencode.ai и залогинься
     2. Нажми F12 → Application → Cookies → opencode.ai
     3. Скопируй значение поля `auth`
     4. Вставь в CodexBar Settings → OpenCode

### MiniMax

1. **Залогинься в браузере** на https://www.minimaxi.com
   - Нужен Chrome или Edge (Brave не поддерживается)
   - Cookies читаются автоматически

2. Если нужна manual авторизация:
   - Settings → MiniMax → вставь `HMACCCS` cookie вручную
   - Как получить cookie:
     1. Открой https://www.minimaxi.com и залогинься
     2. Нажми F12 → Application → Cookies → www.minimaxi.com
     3. Скопируй значение поля `HMACCCS`
     4. Вставь в CodexBar Settings → MiniMax

## Как открыть настройки

1. Кликни на иконку CodexBar в трее (рядом с часами)
2. Нажми **⚙ Settings**
3. Там вкладки: Claude · Z.AI · MiniMax · OpenCode

## Troubleshooting

| Проблема | Решение |
|----------|---------|
| OpenCode показывает "cookies not found" | Залогинься на opencode.ai в Chrome/Edge/Brave, затем перезапусти CodexBar |
| MiniMax показывает "unavailable" | Залогинься на minimaxi.com в Chrome/Edge, затем перезапусти CodexBar |
| Icons in tray нет цифр | Убедись что шрифт `arialbd.ttf` доступен (обычно в `C:\Windows\Fonts\`) |
| Кнопки Test зависают | Обнови версию до 2.2.0 — старые Test кнопки зависали |
| Кнопки Save/Cancel не на месте | Обнови версию — кнопки перемещены вниз окна |

## Проверка что cookies читаются

Открой Settings → нажми Test для нужного провайдера:
- **"Testing..."** → ок, пошло
- **"Valid!"** → credentials рабочие
- **"No cookies found"** → browser cookie не найден, нужен login
- **"Session expired"** → cookie истёк, нужно перелогиниться в браузере

## Ручной cookie — пошагово

### OpenCode:
1. Chrome → https://opencode.ai → login
2. F12 → Application → Cookies → opencode.ai
3. Скопировать значение `auth` (начинается с `Fe26.2**...`)
4. CodexBar → Settings → OpenCode → вставить → Save

### MiniMax:
1. Chrome → https://www.minimaxi.com → login
2. F12 → Application → Cookies → www.minimaxi.com
3. Скопировать значение `HMACCCS`
4. CodexBar → Settings → MiniMax → вставить → Save
