# Тестирование z.ai провайдера

## Для Романа (Windows)

### Шаг 1: Установить зависимости
```cmd
pip install requests customtkinter Pillow pystray
```

### Шаг 2: Скопировать codexbar.py
Заменить файл `codexbar.py` в папке CodexBar-Win на новую версию.

### Шаг 3: Запустить
```cmd
python codexbar.py
```

### Шаг 4: Проверить
- [ ] Таб "Z" виден в popup (рядом с Claude и OpenAI)
- [ ] При пустом ZAI_API_TOKEN — статус "Token not set"
- [ ] Нажать ⚙ (настройки) — появилось поле для Z.AI токена
- [ ] Ввести Z.AI API токен
- [ ] Нажать "Test" → должен появиться ✓
- [ ] Нажать "Save" — настройки сохранены
- [ ] Данные Z.AI отображаются (quota, used %, reset)
- [ ] Progress bar работает
- [ ] **Claude tab** работает как раньше (без изменений)
- [ ] **OpenAI tab** работает как раньше (без изменений)

### Возможные проблемы

| Проблема | Решение |
|----------|---------|
| Таб "Z" не виден | Проверить что логотип `zai-logo.png` лежит рядом с codexbar.py |
| "Token not set" после ввода | Проверить что settings сохраняются (файл `.codexbar_settings.json`) |
| HTTP 401 | Токен невалидный — проверить в z.ai кабинете |
| Timeout | Проверить интернет / прокси |

### Как получить Z.AI API токен
Зайти в https://z.ai → Settings → API Keys → скопировать токен.
