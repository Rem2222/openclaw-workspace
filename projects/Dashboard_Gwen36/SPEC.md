# Спецификация задачи: OpenClaw Dashboard

## Описание задачи

Веб-дашборд для визуализации и управления агентами OpenClaw.

## Цель проекта

Создать веб-интерфейс для мониторинга и управления OpenClaw: агенты, задачи, сессии, cron, activity feed, approvals.

## Ожидаемый результат

Веб-приложение на `http://localhost:3000` с 6 экранами:
1. **Agents** — список агентов, статус, управление
2. **Tasks** — Kanban доска задач
3. **Sessions** — активные сессии
4. **Cron** — планировщик задач
5. **Activity Feed** — лента событий в реальном времени
6. **Approvals** — очередь подтверждений

## Технологии (выбраны)

- **Backend:** Node.js + Express
- **Frontend:** React + Vite
- **UI:** Tailwind CSS
- **Иконки:** Heroicons
- **WebSocket:** socket.io (если доступен)

## Функционал MVP

### 1. Подключение к Gateway
- Выбор инстанса при запуске
- Возможность переключения
- Автосохранение конфига (`~/.openclaw-dashboard/config.json`)
- Тест подключения при старте

### 2. Экран Agents
- Список агентов
- Статус: idle/active/blocked
- Последняя активность
- Кнопки: pause/resume/restart
- Кнопка "Расширенные данные" (CPU, память, задачи)

### 3. Экран Tasks (Kanban)
- Колонки: inbox / assigned / in_progress / review / done / blocked
- Фильтры: по агенту, приоритету (выборочно)
- Задачи перемещаются автоматически при изменении статуса

### 4. Экран Sessions
- Активные сессии
- Модель, токены, context %, время последнего события
- Кнопка "Подробнее" (модальное окно)
- Кнопки управления: kill, clear history

### 5. Экран Cron
- Список задач cron
- Last run, next run, success/fail
- Выключатель с подтверждением (ручной trigger)
- Лог последних N запусков

### 6. Экран Activity Feed
- Realtime лента событий
- Типы: agent message, tool run, status change, errors
- Автообновление каждые 5 секунд (или WebSocket)
- Фильтрация по типу, агенту, уровню
- Пагинация (хранить последние N событий)

### 7. Экран Approvals
- Очередь действий требующих подтверждения
- Кнопки approve/reject
- Автообновление
- Детали approval при клике

### 8. UI
- Тёмная тема по умолчанию
- Адаптивная верстка
- Минималистичный стиль
- Быстрый рендер
- Красивые иконки (Heroicons)
- Плавное переключение между экранами

### 9. Безопасность
- Не показывать секреты
- Localhost по умолчанию
- Токен только в env
- CORS: разрешить все

## Критерии успеха

- [ ] Веб-сервер запущен на порту 3000
- [ ] Подключение к Gateway работает
- [ ] Все 6 экранов отображаются
- [ ] Данные обновляются в реальном времени
- [ ] Управление агентами работает
- [ ] UI минималистичный и понятный
