### Этап 5: Разработка — Backend

**Цель:** Создание Express сервера с API

**Задачи:**
- [ ] Инициализация проекта (npm init, установка зависимостей)
- [ ] Настройка Express сервера
- [ ] Создание API клиента для OpenClaw Gateway
- [ ] Реализация endpoints:
  - `GET /api/agents` — список агентов
  - `GET /api/agents/:id` — детали агента
  - `POST /api/agents/:id/action` — управление агентом
  - `GET /api/tasks` — список задач
  - `GET /api/sessions` — список сессий
  - `GET /api/cron` — список cron задач
  - `GET /api/activity` — лента событий
  - `GET /api/approvals` — очередь подтверждений
  - `POST /api/approvals/:id/approve` — одобрение
  - `POST /api/approvals/:id/reject` — отклонение
- [ ] Настройка CORS
- [ ] Тестирование API

**Агенты:**
- `dev-backend-5-1` — бэкенд разработчик

**Результат:** Рабочий сервер с API

**Время:** 1-1.5 часа

**Зависимости:** Этап 3 (ТЗ готово)

## Параллелизация

### Можно выполнять параллельно:

| Этап | Зависимости | Может запускаться с |
|------|-------------|---------------------|
| Этап 5 (Backend) | Этап 3 | Этап 6 |
| Этап 6 (Frontend) | Этап 3 | Этап 5 |

### Должны выполняться последовательно:

```
Этап 2 → Этап 3 → Этап 4 → (Этап 5 + Этап 6) → Этап 7 → Этап 8 → Этап 9
```
