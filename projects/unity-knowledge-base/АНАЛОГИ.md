# Аналоги unity-knowledge-base

Найденные проекты и скиллы, аналогичные единой системе хранения и поиска знаний (knowledge base), на **ClawHub** и в репозитории **awesome-openclaw-skills**.

---

## 🔍 ClawHub (clawhub.com)

На ClawHub фактически нет отдельной категории «knowledge base» — релевантные скиллы разбросаны по категориям **Notes & PKM**, **Search & Research** и **AI & LLMs**. Сайт ClawHub динамический (SPA на Convex), поэтому контент недоступен через простой web fetch. Все найденные скиллы получены из репозитория awesome-openclaw-skills.

## 📚 awesome-openclaw-skills (github.com/VoltAgent/awesome-openclaw-skills)

### 1. Knowledge Base / PKM (Personal Knowledge Management)

#### 1. `2nd-brain`
- **Ссылка:** https://clawhub.ai/skills/coderaven-2nd-brain
- **Описание:** Personal knowledge base for capturing and retrieving information about people, places, restaurants, games, tech.
- **Чем похож:** Тематическая база знаний по категориям (люди, места, техника) — общая идея структурированного хранения и поиска информации.

#### 2. `brainrepo`
- **Ссылка:** https://clawhub.ai/skills/codezz-brainrepo
- **Описание:** Your personal knowledge repository — capture, organize, and retrieve.
- **Чем похож:** Полный цикл: захват → организация → поиск. Прямой аналог концепции unity-knowledge-base.

#### 3. `books-for-agents`
- **Ссылка:** https://clawhub.ai/skills/danpalmieri-books-for-agents
- **Описание:** An open source knowledge base of structured book summaries, available to any AI agent via MCP.
- **Чем похож:** Централизованная база структурированных знаний (книжные выжимки), доступная AI-агентам.

#### 4. `1` (Ensue-powered KB)
- **Ссылка:** https://clawhub.ai/skills/nastrology-1
- **Описание:** Personal knowledge base powered by Ensue for capturing and retrieving.
- **Чем похож:** Персональная база знаний с захватом и поиском — близкая концепция.

#### 5. `anshumanbh-qmd`
- **Ссылка:** https://clawhub.ai/skills/anshumanbh-anshumanbh-qmd
- **Описание:** Search markdown knowledge bases efficiently.
- **Чем похож:** Поиск по markdown-базам знаний — аналогичная задача индексации и поиска.

### 3. Vector / Semantic Memory

#### 13. `aister-vector-memory`
- **Ссылка:** https://clawhub.ai/skills/alekhm-aister-vector-memory
- **Описание:** Vector memory for Aister — search by meaning, not by grep!
- **Чем похож:** Векторная память с поиском по смыслу (semantic search) — прямое применение эмбеддингов для поиска.

#### 14. `enhanced-memory`
- **Ссылка:** https://clawhub.ai/skills/jameseball-enhanced-memory
- **Описание:** Enhanced memory search with hybrid vector+keyword scoring, temporal routing, filepath scoring, adaptive weighting.
- **Чем похож:** Гибридный поиск (векторный + ключевые слова) — аналогичная задача повышения качества поиска.

#### 15. `engram-memory`
- **Ссылка:** https://clawhub.ai/skills/dannydvm-engram-memory
- **Описание:** Persistent semantic memory for AI agents — local, fast, free.
- **Чем похож:** Персистентная семантическая память — хранение и поиск знаний по смыслу.
